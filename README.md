# Enhancing Large Language Model Workflows for Gene Prioritization

Overview : 
This project focuses on improving gene prioritization using Large Language Models (LLMs) by integrating biomedical datasets such as DisGeNET, PubMed, GEO, and STRING. The goal is to enhance the ranking and selection of disease-related genes using AI-driven text understanding and biological knowledge integration.

Objectives : 
Improve gene prioritization using LLM-based workflows
Integrate multiple biomedical data sources
Extract meaningful gene-disease relationships
Rank genes based on relevance and biological significance
Support research in genomics and computational biology

Key Features : 
LLM-based gene relevance analysis
Biomedical dataset integration (DisGeNET, PubMed, GEO, STRING)
Gene-disease relationship extraction
Prioritization and ranking system
Scalable workflow for research applications

Tech Stack : 
Python
Machine Learning / NLP
Large Language Models (LLMs)
Biomedical Databases
Data Processing Libraries (Pandas, NumPy)

Workflow : 
Collect biomedical data from multiple sources
Process and clean gene-related information
Use LLMs to analyze gene-disease relationships
Score and rank genes based on relevance
Output prioritized gene list for research use

 Use Cases : 
Disease gene discovery
Biomedical research support
Drug target identification
Genomics data analysis
AI-assisted biological research
